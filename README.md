# 🕵️‍♂️ Detective de Datos: Descubriendo Patrones Ocultos

Proyecto de Minería de Datos para un supermercado, diseñado para descubrir patrones de compra, segmentar clientes y detectar transacciones sospechosas.

---

## 🚀 Tecnologías
- Python  
- NumPy / Pandas  
- Scikit-learn  
- Mlxtend (Apriori, Reglas de Asociación)  

---

## 📊 Análisis implementados

- **Análisis de Canasta de Compra** (Reglas de Asociación)  
- **Segmentación de Clientes** (KMeans)  
- **Detección de Anomalías** (Transacciones raras)  

</details>

---

## ▶️ Ejecución
```bash
pip install numpy pandas scikit-learn matplotlib
pip install mlxtend

python detective_datos.py
```
